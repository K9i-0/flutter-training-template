# 天気予報を取得し、表示する

[yumemi_weather] の API を使用して、天気予報を取得しましょう。

## 課題

- Reload ボタンをタップして天気予報を取得する
- 天気予報を画面に表示する

### 使用する API

- `fetchSimpleWeather()`

### 利用する画像

- [sunny]
- [cloudy]
- [rainy]
